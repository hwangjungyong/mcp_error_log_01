## 주요 변경 사항

### 1. 포트 설정
- **API 서버**: 3001 → **3011** (충돌 방지)
- **Vite 개발 서버**: 5173 → **5183** (충돌 방지)

### 2. 기능 범위
- ✅ AI 에러로그분석
- ✅ AI 에러로그 현황
- ❌ 인증 기능 (제거)
- ❌ 기타 기능 (제거)

### 3. 데이터베이스
- 새로운 SQLite 데이터베이스 생성
- 테이블: `error_logs`, `error_log_metadata`
- 위치: `data/database.db` (자동 생성)

### 4. API 엔드포인트
- `POST /api/error-log/analyze`: 에러 로그 분석
- `POST /api/error-log/save`: 에러 로그 저장
- `GET /api/error-log/history`: 에러 로그 이력 조회

## 다음 단계: 실행 방법

### 1. 환경 변수 설정 (필수)

`.env` 파일을 프로젝트 루트(`C:\test\mcp_error_log_01`)에 생성:

```env
# API 서버 설정
PORT=3011

# Vite 개발 서버 포트
VITE_PORT=5183

# API Base URL
VITE_API_BASE_URL=http://localhost:3011
```

**생성 방법 (PowerShell):**
```powershell
cd C:\test\mcp_error_log_01
@"
# API 서버 설정
PORT=3011

# Vite 개발 서버 포트
VITE_PORT=5183

# API Base URL
VITE_API_BASE_URL=http://localhost:3011
"@ | Out-File -FilePath .env -Encoding UTF8
```

### 2. 의존성 설치

```bash
cd C:\test\mcp_error_log_01

# Node.js 패키지 설치
npm install

# Python 패키지 설치
pip install mcp anthropic
```

### 3. 실행

```bash
# 방법 1: 통합 실행 (권장)
npm start

# 방법 2: 개별 실행
# 터미널 1
npm run api-server

# 터미널 2
npm run dev
```

### 4. 접속

- **프론트엔드**: http://localhost:5183
- **API 서버**: http://localhost:3011
